## Key Decisions Made

| ADR | Decision | Rationale |
|-----|----------|-----------|
| ADR-004 | Web UI over desktop app | Covers local + cloud hosting, no binary distribution |
| ADR-005 | TypeScript for server | Best Docker SDK, shared types with UI, strong typing |
| ADR-006 | pnpm monorepo | Shared types, atomic commits, single CI pipeline |
| ADR-007 | Environment snapshots | Eliminates cold-start, <5s boot from pre-built images |
| ADR-008 | SQLite knowledge system | Cross-session memory, zero-config, self-hosted friendly |
| ADR-009 | Named greeting protocol | Anti-hallucination anchoring, forces context grounding |
